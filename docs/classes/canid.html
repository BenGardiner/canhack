CANID — CAN identifier¶

class CANID(canid[, extended=False])¶

Represents a CAN identifier.

Parameters

  • canid (int) – The CAN identifier of the frame

  • extended – Set to True if the ID is a 29-bit identifier.

Raises

ValueError – if canid is not in the range 0..0x7ff (when extended is False) or 0..0x1fffffff (when extended is True).

A CANID instance is created either directly or is returned from the get_canid() call.

Methods¶
