This document describes various software and hardware CAN projects that together are called the Yes we CAN collection. The projects include:

  • The Canis CANHack board for the Raspberry Pi Pico

  • The Canis CANPico board for the Raspberry Pi Pico

  • A MicroPython CAN API for the Raspberry Pi Pico and Canis CANPico board

  • A MicroPython CANHack toolkit API for the Raspberry Pi Pico with the Canis CANHack board or the Canis CANPico board

The rest of this manual documents these projects.
